## REQUIRED API ENDPOINTS (Backend)

```
GET /api/student/{studentId}
  Response: { exists: bool, name: string, lastPredictionDate: string, data: {...} }

POST /api/integrations/leetcode
  Body: { username: string }
  Response: { score: number, problemsSolved: number, ... }

POST /api/integrations/github
  Body: { username: string }
  Response: { score: number, repositories: number, ... }

POST /api/integrations/resume-ats
  Body: FormData { resume: File }
  Response: { atsScore: number, keywordsFound: [...], ... }

POST /api/predictions/generate
  Body: {
    studentId: number,
    dsa_score: number,
    project_score: number,
    cs_fundamentals_score: number,
    aptitude_score: number,
    hr_score: number,
    resume_ats_score: number,
    hackathon_wins: number,
    cgpa: number
  }
  Response: {
    overall_placement_probability: number,
    predicted_salary_lpa: number,
    salary_range_min_lpa: number,
    salary_range_mid_lpa: number,
    salary_range_max_lpa: number,
    prob_salary_gt_2_lpa: number,
    prob_salary_gt_5_lpa: number,
    prob_salary_gt_10_lpa: number,
    prob_salary_gt_15_lpa: number,
    prob_salary_gt_20_lpa: number,
    prob_salary_gt_25_lpa: number,
    prob_salary_gt_30_lpa: number,
    prob_salary_gt_35_lpa: number,
    prob_salary_gt_40_lpa: number,
    predicted_job_role: string,
    recommended_companies: [string, ...]
  }

POST /api/predictions/save
  Body: { studentId: number, predictions: {...} }
  Response: { success: bool, message: string }
```

## ERROR HANDLING

**Handle These Cases:**
- Invalid Student ID: "Student not found. Please check and try again."
- LeetCode not found: "Username not found. Please enter score manually."
- GitHub not found: "Username not found. Please enter score manually."
- Resume upload failed: "File upload failed. Please try again."
- API timeout: "Connection timeout. Please check your network."
- Validation error: "Please fill all required fields with valid values."
- Prediction failed: "Could not generate predictions. Try again later."

**UI Behavior:**
- Show error messages inline near the field
- Highlight invalid fields                                                                                                            
- Disable Submit/Next button until fixed
- Show retry button for API failures
